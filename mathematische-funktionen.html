Mathematische Funktionen

Funktionen sind grundlegende Konzepte in der Mathematik, die Beziehungen zwischen Variablen beschreiben.

Einführung: Mathematische Funktionen

Was ist eine Funktion?

Eine Funktion ist eine Art „Rechenvorschrift“, die jedem Wert aus einer Menge (meistens eine Zahl, genannt x) genau einen Wert aus einer anderen Menge (meistens eine Zahl, genannt y oder f(x)) zuordnet.

Du kannst dir eine Funktion wie eine Maschine vorstellen:

  • Du steckst eine Zahl hinein (x).
  • Die Maschine rechnet damit und spuckt eine neue Zahl aus (f(x)).

Warum sind Funktionen wichtig?

🔍 Sie helfen uns, Zusammenhänge zu verstehen
Mit Funktionen kannst du mathematische Zusammenhänge darstellen — zum Beispiel, wie sich eine Grösse verändert, wenn eine andere grösser wird.

📈 Sie tauchen überall auf
Funktionen begegnen dir in Physik, Chemie, Wirtschaft, Informatik — und natürlich in der Mathematik.

Grundlagen zu Funktionen

👉 Jede Funktion besteht aus:

  • Eingabe (x) — der Wert, den du einsetzt.
  • Ausgabe (f(x)) — der Wert, den die Funktion berechnet.

Ein Beispiel:

f(x) = 2x + 3

Wenn du x = 5 einsetzt, rechnest du:

f(5) = 2⋅5 + 3 = 13

Beispiele

Hier ein paar typische Funktionen, die dir im Unterricht begegnen:

  • ✅ Lineare Funktion: f(x) = 2x + 3
    Diese Funktion wächst gleichmässig: Wenn x grösser wird, wächst f(x) ebenfalls linear.
  • ✅ Quadratische Funktion: g(x) = x² - 4x + 1
    Hier ist der höchste Exponent von x eine 2. Diese Funktion bildet eine Parabel, die nach oben geöffnet ist.
  • ✅ Wurzelfunktion: h(x) = √x
    Hier darf x nur grösser oder gleich 0 sein, weil man keine Wurzel aus einer negativen Zahl ziehen kann.
